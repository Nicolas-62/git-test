Et bien, si vous ne faites que mettre à jour un fichier que vous aviez déjà ajouté à l'index,
vous pouvez condenser ces deux étapes (git add et git commit) de la façon suivante :

git commit -a -m "Ajouté itinéraire dans checklist-vacances.md"

supprimer modifs depuis dernier commit :

git reset --hard HEAD ou git reset --hard

supprimer dernier commit :

git reset --hard HEAD^

Ajout de code pour fusion des branches

se placer sur master et faire :

git merge branche

résolution conflits

lors de la fusion ne garder qu le code voulu dans les zones de conflit puis faire un commit sans titre

voir qui à édité le code dans un fichier :

git blame nomdufichier.ext

voir le code du fichier à un commit donné :

git show debutducommit
